# Priyansh Portfolio 🚀

A modern, Apple-inspired personal portfolio website built for **Priyansh Jain** — a web developer specializing in creating fast, scalable, and visually stunning web applications.

---

## ✨ Features

- 🍎 **Apple-style dark theme** — clean, minimal, premium aesthetic
- 🎬 **Smooth animations** — fade-in on load, scroll-triggered reveals, floating hero text
- 🖱️ **Hover effects** — subtle micro-interactions on all buttons and elements
- 📱 **Fully responsive** — works on desktop, tablet, and mobile
- ⚡ **Fast & lightweight** — pure HTML, CSS, JavaScript (no heavy frameworks)
- 📬 **Contact section** — direct email link for easy outreach

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/priyansh-portfolio.git
   ```

2. **Open the project**
   ```bash
   cd priyansh-portfolio
   ```

3. **Run with Live Server**
   - Open in VS Code
   - Right click `index.html` → **Open with Live Server**

   Or simply open `index.html` directly in your browser.
